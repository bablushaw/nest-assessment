## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Git

```bash
git clone https://github.com/bablushaw/nest-assessment.git

cd nest-assessment
```

## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev
```

## Run tests

```bash
# unit tests
$ npm run test
```
